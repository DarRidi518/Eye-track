# Eye-track - проект, который сократит количество аварий

# Что?
  Программа, которая отслеживает состояние глаз во время поездки, и в опасных случаях предотвращает аварии

# Зачем?
  Создать встроенный в машину механизм, который будет следить за глазами водителя. В случае их длительного закрытого положения вывести предупреждение о неактивном состоянии водителя. В случае игнорирования предупреждения аккуратно остановить движение машины, чтобы не произвести аварии. Это сократит количество аварий примерно на 30-35%

# Где?
  Продукт не будет продаваться отдельно. Продукт будет новой функцией, присущей какой-либо марке машин

# Когда?
  Продукт постоянно нужный, без исключения

# Кому?
  Продукт понадобится каждому водителю, с проблемами, которые могут вызвать бессознательное состояние за рулем, что может привести к ДТП на дороге

Язык программирования я выбрал python, потому что он интуитивно понятен и относительно быстр
