# Eye-track - проект, который сократит количество аварий

# Что?
  Программа, которая отслеживает состояние глаз водителя во время поездки, и в опасных случаях предотвращает аварии

# Зачем?
  Создать встроенный в машину механизм, который будет следить за глазами водителя. В случае их длительного закрытого положения вывести предупреждение о неактивном состоянии водителя. В случае игнорирования предупреждения аккуратно остановить движение машины, чтобы не произвести аварии. Это сократит количество аварий примерно на 30-35%

# Где?
  Программа не будет продаваться отдельно. Это будет новой функцией, присущей какой-либо марке машин

# Когда?
  Постоянно нужная функция, без исключения

# Кому?
  Функция контроля состояния водителя понадобится каждому, кто подвержен проблемам, которые могут привести к потере сознания за рулём и ДТП

Язык программирования я выбрал python, потому что он легок (в сравнении с C++) и относительно быстр (в сравнении с Java или SQL)

Экономика B2B, потому что у меня нет своей марки машин чтобы добавить ее туда
